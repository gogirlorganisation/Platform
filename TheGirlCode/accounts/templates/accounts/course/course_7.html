Functions · Python

In Python, function is a group of related statements that perform a specific task.
Functions help break our program into smaller and modular chunks.(How does that help?) As our program grows larger and larger, functions make it more organized and manageable. Furthermore, it avoids repetition and makes code reusable.

Function blocks begin with the keyword def followed by the function name and parentheses ( ( ) ).

Syntax:
def function_name()

You have used the function len() earlier, which is a built-in function (These functions are pre-defined in the language and thus we don;t need to tell the compiler hwat len() does, we just need to use it) that returns the length of the string entered.

Some more built in functions you’ve learnt are, print(), .upper() and all the functions you used to move your turtle!

But what if you want to create a function for something that doesn’t have a predefined meaning? There exist user-defined functions which you (i.e the user) can create.

The first step is to define your function, give it a name and parameter: let’s talk about Mathematics, in Trigonometry you have functions (sin x, cos x, tan x), now, they do not make sense at all if a value for x isn’t given. Here x is our parameter and that’s exactly what we need in coding too!

So let’s define a function add that adds two numbers given as parameters

def add(m, n):
   return m + n

print add(123, 77)

#calling the function Once you call the function (use the function in your program), the parameters provided within the brackets are used to carry it out(in this example, we carry out an addition function). The variable m will have the value of 123, n of 77 and the function yields their sum.

**Note : That colon present after the bracket is important, do not forget to add it after defining a function, else there will be an error.

We can create a function using strings too.

def statement(name)
   print (name + “, you’re doing amazing sweetie”)

Calling a function

After defining a function, in order to use it you need to call it. This means that you refer to the function you created in your code in order to use it. To call a function in Python, use the function name, brackets (opening and closing) and parameter(s).

Example:

def avgnum(x, y):
   print("Average of ",x," and ",y, " is ",(x+y)/2)

avgnum(3, 4) #calling a function.

Return

The return statement in a function is used to exit it and go back to the place from where it was called.

return [expression_list]

This statement can contain expression which gets evaluated and the value is returned. If there is no expression in the statement or the return statement itself is not present inside a function, then the function will return the None object.

For example:

def absolute_value(num):
#This function returns the absolute
#value of the entered number

if num >= 0:
return num
else:
return -num

Here, if number is greater than or equal to 0, the number as it is returned , otherwise the number is returned with a negative sign in front of it.
