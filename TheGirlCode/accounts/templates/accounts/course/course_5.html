Turtle · Python

Turtle Graphics Want to bring your programs to life?
You can do so easily, with a little something called Turtle .

Python has a special module called turtle that we can use to learn how computers draw pictures on a screen. The turtle module is a way of programming vector graphics, which is basically just drawing with simple lines, dots, and curves.

To use turtle you need to install turtle on your compiler, which can be done easily with this simple command:

import turtle

You must copy - paste this code onto your compiler to get started. Now it’s time to name your turtle!

eve = turtle.Turtle()

Guess what?
You can now use functions to move your turtle Eve!

eve.forward()
eve.backward()
eve.right()
eve.left()

eve.forward(50)

The turtle has moved forward 50 pixels. A pixel (is a single point on the screen—the smallest element that can be represented. Everything you see on your computer monitor is made up of pixels, which are tiny, square dots.)
To erase the canvas, enter reset. This clears the canvas and puts the turtle back at its starting position.

eve.reset()

You can also use clear, which just clears the screen and leaves the turtle where it is.

eve.clear()

You can change the colour of your turtle:

eve.color("blue")

Let's make a square!

import turtle
eve = turtle.Turtle()
eve.color("blue")

eve.forward(50)
eve.left(90)
eve.forward(50)
eve.left(90)
eve.forward(50)
eve.left(90)
eve.forward(50)

Ughhh, repeating so much code is never good. Let's apply a loop here!

eve.forward(50)
eve.left(90)

The block of code above is being repeated 4 times to create a square. Let's apply a for loop here.

import turtle
eve = turtle.Turtle()
eve.color("blue")

for x in range(0,4): #repeats the code block 4 times
   eve.forward(50)
   eve.left(90)

Great!! Now you are ready to create anything that you want! Go on and explore!
