Conditional Flow · Python

Is there anything worse than getting soaked in the cold rain?
Eve certainly doesn’t think so.
Should she take an umbrella with her when she leaves the house? The logical solution to Eve’s dilemma would be-

if it_is_raining :
   carry_an_umbrella
else:
   do_not_carry_an_umbrella

Similarly, let's tackle another problem. If a child is older than 12, we usually refer to them as teenagers. Let’s create a program for that.

Take the user’s age as the input and save it in a variable age .

age = input("Enter age")

If age is more than 12
   print(“Welcome to the teen years!”)

    Here we can use conditions like:
  • == Equal to
  • ! = Not equal to
  • > Greater than
  • < Less than

So,

if age > 12 :

   print ("Welcome to the teen years!")

**Note: A colon i.e. ' : ' has to be added after every if condition. If the colon is not placed after the condition, there will be a syntax error in your program. Also, the space on the second line is part of the Python Syntax.

Syntax:

if <"some condition"> :

   action

Want to know if you have enough chocolates or not? Simple! Set a condition. If you have more than 25 chocolates, you’re Willy Wonka!

Number_of_choc > 25:

   print("Greetings,Willy Wonka!")


But wait..what if you wish to inform the user that they don't have enough chocolates? Well, you have else to the rescue!

if Number_of_choc > 25:

   print("Greetings,Willy Wonka!")

else:

   print("Not everyone can be a Willy Wonka!")
