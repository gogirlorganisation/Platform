Strings · Python

You’ll use strings to display messages in your programs (such as “Game Over” messages in a game).

In programming terms, we usually call text a string.

var1 = "Taylor Swift is amazing!!"

Then, to see what’s inside var1, we could enter

print(var1).

Output: Taylor Swift is amazing!!

Go on! Try it yourself!

Length of a string

What if you want to know the length of your string? Like Hello contains 5 letters.

Instead of counting each letter, all you need to do is use len()

len(“Hello”)

Output: 5

Or you could use it with variables!

greetings = “Hello”

len(greetings)

String Concatenation

You can easily add strings to build a complete sentence by using the addition operator-ie the ‘+’ . Crazy, right?

P.S. no using this in your english paper! We don’t want your English teacher to faint :P

Now,let’s frame a greeting, shall we?

name = “Eve”
print “Good” + “ afternoon” + name + “!”

OUTPUT : Good afternoon Eve!

**Note: name is not inside "" as it's not a string but a variable!

Try creating a concatenation of the list of things you ate for breakfast! If you had milk and cereal then,

print “Milk” + “ and” + “ cereal”

Do not forget to add space before each word, or your sentence would be one word instead , Milkandcereal and not Milk and cereal



Accessing String Elements

What if you want to access a particular letter in a string? Don't worry, all that you need to do is:

var = "HELLO"

var[2] # access the 2nd element of Hello

print(var)

Guess what will be printed?

E?

Wrong!

There is a catch, programmers like counting from 0, so


Thus, L is printed!

Output: L

The postition alloted to each letter is known as it's index, thus this method is known as indexing.

String slicing

Want a part of your string? No worries!

String[start_position:end_position]

To get "Love" out of "Ada Lovelace" you need letters from the postion 4 to 8.



So, simply print:

print("Ada Lovelace"[4:8])

Output: Love **Note: In programming languages, the end postion is the number after the required number.

Lower and Upper

To change a string to lower case or upper case simply use string.upper() or string.lower().

name = "Eve"
name_upper = name.upper()
name_lower = name.lower()

Now you can simply print them!

print(name)

Output: Eve

print(name_upper)

Output: EVE

print(name_lower)

Output: eve
