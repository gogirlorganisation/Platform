Today, we’ll learn how to apply the concepts you’ve learnt to create a calculator. To make a calculator, you’ll need to be familiar with the following concepts:
Functions
Operators

To make a calculator that adds, subtracts, multiplies and divides you’ll need to define 4 functions that add, subtract , multiply and divide respectively.

# This function adds two numbers
def add(x, y):
return x + y
# This function subtracts two numbers
def subtract(x, y):
return x - y
# This function multiplies two numbers
def multiply(x, y):
return x * y
# This function divides two numbers
def divide(x, y):
return x / y

Then we have to provide the user with options. After entering two numbers, they need to enter an option to decide which function they wish to perform.
print("Select operation.")
print("1.Add")
print("2.Subtract")
print("3.Multiply")
print("4.Divide")
Once you have printed the options you need the input from the user.

# Take input from the user
choice = input("Enter choice(1/2/3/4):")

Then you need the two numbers.

num1 = input("Enter first number: ")
num2 = input("Enter second number: ")
After you have all the input you need to act accordingly.

if choice == '1':
   print(num1,"+",num2,"=", add(num1,num2))
elif choice == '2':
   print(num1,"-",num2,"=", subtract(num1,num2))
elif choice == '3':
   print(num1,"*",num2,"=", multiply(num1,num2))
elif choice == '4':
   print(num1,"/",num2,"=", divide(num1,num2))

For the case when the user doesn't enter a valid choice make an else statement

else:
   print("Invalid input")
