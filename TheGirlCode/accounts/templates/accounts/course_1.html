A computer program is the set of instructions that tell that the dumb hardware (mouse,keyboard,desktop-the physical parts of a computer) what to do.

A programming language is simply a means to communicate with the computer—a way to use instructions that the human and the computer can understand. Python is one such language!

Programming is the process of taking an algorithm (A programming algorithm is a a lot like a recipe - the ingredients are called the inputs and the dish is the output which tells your computer precisely what steps to take to solve a problem.) and writing it in a language that the computer understands.

Variables

Just like your brain, the computer also has memory where it stores data (pieces of information).
A variable can also be called a label for something.
Let’s create a variable red and tell Python that it labels the number 100 ( this doesn’t mean that another variable can’t have the same value):
red = 100
Now to display what the variable contains, type:
print (red)
Output: 100
We can also use more than one label (more than one variable) for the same item:
red = 200
blue = red
print(blue)
Output: 200
Variable names can be made up of:

  • Letters and Numbers ( eg: var1, var45)
  • the underscore character (_)

    • but they can’t start with a number
