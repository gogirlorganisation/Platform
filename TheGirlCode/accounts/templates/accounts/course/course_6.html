Lists· Python

Eve wants to go grocery shopping, so, she makes a list of all the items she needs. Her list contains the following items:

  • Butter
  • Bread
  • Chocolate
  • Carrots

Python offers a tool called lists to keep track of related "things," or values which are similar.
To help Eve, let's make a list called shopping_list and fill it with each of the items:



Programmers like to count from 0. Hence, instead of having 1, 2, 3 and, 4 position numbers, we have 0, 1, 2 and, 3 position numbers!

shopping_list = [“carrots", "chocolate", "butter", "bread"]

If we want to print the third item in the shopping_list (butter) we can do so by entering its position in the list (called the index position) inside square brackets ([]), like we did for strings:

print (shopping_list[2])

Output: butter

What if we want to print all the elements?

for x in range(0,4):
   print (shopping_list[x])

OR

print shopping_list

Eve forgot to add flour to her list! Oh no, how will she bake cake now?
But, fear not..you can add flour to her shopping list since there’s no limit for the length of the shopping list.

Syntax:
list_name.append(item)

shopping_list.append(“flour”)

Now using append(), the item can only be added to the end of the list. But do not fret, we can add the item to whichever position you like using another function insert(). Eve wants to buys toffees! Let's put toffees after carrots. So we to insert it at index number 1.

Syntax:
list_name.insert(index number, item)

shopping_list.insert(1,"toffees")

Within the brackets first mention the desired index number and then the item.

Sometimes you need to remove something from the list, that can achieved by yet another function remove().
Silly Eve forgot she already had a loaf of bread at home, so she needs our help to remove that item from the list.

Syntax:
list_name.remove(item)

shopping_list.remove(“bread”)

Eve has realised that she doesn't need carrots for baking a cake! She wants to buy eggs now... Easy! Just replace the carrots!! Carrots are on the first position, so their index number is 0.

Syntax:
list_name[postion_of_the_replacement] = new_value

shopping_list[0] = "eggs"

Good work! Now Eve can go shopping!!
