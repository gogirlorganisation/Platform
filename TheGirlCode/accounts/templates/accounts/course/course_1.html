A computer program is the set of instructions that tell that the dumb hardware (mouse,keyboard,desktop-the physical parts of a computer) what to do.

A programming language is simply a means to communicate with the computer—a way to use instructions that the human and the computer can understand. Python is one such language!

A computer program is made of algorithms. (An algorithm is like a recipe: if you’ve ever cooked, you probably know that you are made to follow certain steps. Say, you want to make a glass of coffee. You would first put the coffee powder, then pour the milk, and then stir. For computers, this step-by-step instruction is called an Algorithm.)

Programmers use comments to explain what a section of the code does for someone who’s viewing the program, so even the layman can understand what this ‘coding’ is all about. It's written after the symbol // .

print("Hello World") // this is a comment

The sad part is that computers don’t understand English or Hindi – they have their own languages. When an algorithm is written in one of these languages, it becomes a Program. This is what you’ll learn over the next few days!

Operators

The basic symbols used by Python to perform mathematical operations are called operators:

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division for Quotient (/)
  • Modulo (%)

What does Modulo do, you may ask? Well it’s quite simple.
Try figuring out what it does:
1 % 4 = 1
15 % 4 = 3
10 % 4 = 2

Got it? Yes, you’re right! Modulo returns the remainder.
4 x 0 + 1
4 x 3 + 3
4 x 2 + 2
Good work!

Python follows BODMAS rules. For example, if you enter
5+6 *10
5 and 10 are multiplied and 6 is added to the result- the answer is 56 and not 110.

We use brackets () to correct this. The bracket tells Python to perform the operation inside it first.
( 5+6) *10= 110.
Here, the operation inside the brackets is performed first, so 11 is multiplied to 10.

The < sign means "less than," and the > sign "greater than" in Python.
boolean operators

Variables

Just like your brain, the computer also has memory where it stores data (pieces of information).

A variable is used to store information. It also be called a label for something.

Let’s create a variable red and tell Python that it labels the number 100 (this doesn’t mean that another variable can’t have the same value):

red = 100

Now to display what the variable contains, type:

print (red)

Output: 100

We can also use more than one label (more than one variable) for the same item:

red = 200

blue = red

print(blue)

Output: 200

Variable names can be made up of:

  • Letters and Numbers ( eg: var1, var45)
  • the underscore character (_)
But remember that they can’t start with a number

The numbers stored are a special type of data i.e integer (negative and positive numbers without a decimal point eg: -5, 100, 0).
