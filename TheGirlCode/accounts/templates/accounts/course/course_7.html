Lists· Python

Eve wants to go grocery shopping, so, she makes a list of all the items she needs. Her list contains the following items:

  1. Butter
  2. Bread
  3. Chocolate
  4. Carrots


    5. Python offers a tool called lists to keep track of related "things," or values.
    To help Eve,let's make a list called shopping_list and fill it with each of the items:
    PICTURE LIST 0-3 items
    Programmers like to count from 0. Hence, instead of having 1, 2, 3 and, 4 position numbers, we have 0, 1, 2 and, 3 position numbers!

    shopping_list = [“bread", "butter", carrots", "chocolate"]

    If we want to print the third item in the shopping_list (carrots) we can do so by entering its position in the list (called the index position) inside square brackets ([]), like this:

    print (shopping_list[2])

    shopping_list = [“bread", "butter", carrots", "chocolate"]

    For x in range(0,3):
    Print (shopping_list[x])

    Eve forgot to add flour to her list! Oh no, however will she bake cake now? But, fear not..you can add flour to her shopping list since there’s no limit for the length of the shopping list.
    The syntax for appendment is:
    list_name.append(item) shopping_list.append(“flour”) print shopping_list

    Now using append(), the item can only be added to the end of the list. But do not fret, we can add the item to whichever position you like using another function insert()

    list_name.insert(index number, item

    shopping_list.insert(1,”flour)
    print shopping_list

    Within the brackets first mention the desired index number and then the item.
    Sometimes you need to remove something from the list, that can achieved by yet another function remove().
    Silly Eve forgot she already had a loaf of bread at home, so she needs our help to remove that item from the list.

    list_name.remove(item)
    OR
    shopping_list.remove(“bread”)
    print shopping_list
