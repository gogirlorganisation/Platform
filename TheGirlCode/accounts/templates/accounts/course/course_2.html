Variables · Python

Just like your brain, the computer also has memory where it stores data (pieces of information).

A variable is used to store information. It also be called a label for something.

Let’s create a variable red and tell Python that it labels the number 100 (this doesn’t mean that another variable can’t have the same value):

red = 100

Now to display what the variable contains, type:

print (red)

Output: 100

We can also use more than one label (more than one variable) for the same item:

red = 200

blue = red

print(blue)

Output: 200

Variable names can be made up of:

  • Letters and Numbers ( eg: var1, var45)
  • the underscore character (_)
But remember that they can’t start with a number

Go on and make a variable year of value 2018.

The numbers stored are a special type of data i.e integer (negative and positive numbers without a decimal point eg: -5, 100, 0).

You’ll use strings to display messages in your programs (such as “Game Over” messages in a game).

In programming terms, we usually call text a string.

var1 = "Taylor Swift is amazing!!"

Then, to see what’s inside var1, we could enter

print(var1).

Output: Taylor Swift is amazing!!

Go on! Try it yourself!

Length of a string

What if you want to know the length of your string? Like Hello contains 5 letters.

PICTURE OF HELLO WITH 1-5 LABELLED

Instead of counting each letter , all you need to do is use len()

len(“Hello”)

Output: 5

Or you could use it with variables!

greetings = “Hello”

len(greetings)

String Concatenation

You can easily add strings to build a complete sentence by using the addition operator-ie the ‘+’ . Crazy, right?

P.S. no using this in your english paper! We don’t want your English teacher to faint :P

Now,let’s frame a greeting, shall we?

name = “Eve”
print “Good” + “ afternoon” + name + “!”

OUTPUT : Good afternoon Eve!

Try creating a concatenation of the list of things you ate for breakfast! If you had milk and cereal then,

print “Milk” + “ and” + “ cereal”

Do not forget to add space before each word, or your sentence would be one word instead , Milkandcereal and not Milk and cereal

print(var)

Print

Print has been used multiple times till now. It is the command you use to tell the computer to display something.
For example, let’s display Hello World on our screen. To do this,

print (Hello World)

print function helps you display the code on the output screen.

You can directly print strings like

print(“I am amazing”)

or simply display a variable, like this

print(variable)
